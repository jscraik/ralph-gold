# Support

Doc requirements:
- Audience: users
- Scope: where to ask questions and report problems
- Owner: maintainers
- Review cadence: quarterly
- Last updated: 2026-01-15

## How to get help

- Use GitHub Issues for bug reports and feature requests.
- For security issues, use the contact in `SECURITY.md`.

## What to include in an issue

- CLI version (`ralph --version`)
- Operating system and shell
- The command you ran
- The error output (redact secrets)
